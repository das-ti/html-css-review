## Semana 1 — HTML semântico + CSS base

### Segunda-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | Tags semânticas: `header`, `nav`, `main`, `section`, `article`, `footer`. Diferença entre `div` e tags semânticas. |
| 💻 **Noite** | Criar estrutura HTML de uma página de portfólio pessoal (só estrutura, sem CSS ainda). Commit: `study: estrutura html da pagina de portfolio` |

### Terça-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | CSS: seletores, especificidade, box model, unidades (px, rem, %). Cores e tipografia. |
| 💻 **Noite** | Adicionar CSS básico na página: fontes, cores, margens. Commit: `style: adiciona css base ao portfolio` |

### Quarta-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | Flexbox completo: `display: flex`, `direction`, `justify-content`, `align-items`, `gap`, `wrap`. |
| 💻 **Noite** | Criar navbar e seção hero com Flexbox. Commit: `style: layout do header com flexbox` |

### Quinta-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | CSS Grid: `grid-template-columns`, `rows`, `areas`, `gap`. Quando usar Grid vs Flexbox. |
| 💻 **Noite** | Seção de cards de projetos com Grid. Commit: `style: secao de projetos com css grid` |

### Sexta-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | Formulários HTML: `input`, `select`, `textarea`, `button`, `label`, validação nativa. Estilização com CSS. |
| 💻 **Noite** | Seção de contato com formulário. Commit: `feat: formulario de contato estilizado` |

## Semana 2 — Responsividade, variáveis CSS e transições

### Segunda-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | Media queries, breakpoints, mobile-first. Adaptação de layouts para diferentes telas. |
| 💻 **Noite** | Tornar a página responsiva para mobile. Commit: `style: responsividade mobile no portfolio` |

### Terça-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | Variáveis CSS (custom properties). Pseudo-classes `:hover`, `:focus`, `:nth-child`. Pseudo-elementos `::before`, `::after`. |
| 💻 **Noite** | Adicionar variáveis de cor e efeitos hover. Commit: `style: variaveis css e hover nos cards` |

### Quarta-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | `transition` e `animation` CSS. Micro-interações sem JavaScript. |
| 💻 **Noite** | Animar navbar, cards e botões. Mini: loading spinner em CSS puro. Commit: `style: transicoes e animacoes no portfolio` |

### Quinta-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | `position`: relative, absolute, fixed, sticky. `z-index`. Dropdowns e overlays com CSS. |
| 💻 **Noite** | Menu dropdown com CSS puro. Commit: `feat: menu dropdown com css position` |

### Sexta-feira
| Período | Conteúdo |
|---|---|
| 📖 **Tarde** | Revisão geral. Análise de um desafio do Frontend Mentor como referência de layout. |
| 💻 **Noite** | Finalizar e polir a página de portfólio. Publicar no **GitHub Pages**. Commit: `feat: pagina de portfolio finalizada` |
